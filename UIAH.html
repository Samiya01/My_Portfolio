A Website & mobile App where farmers can log there problem through
voice or text message & they will get solutions, suggestions &
image description of the solution. It has a chat-like interface where
the farmers can get immediate responses with image description the
responses come in both audio & text from enabling the user to listen
& or read the responses. And also farmer can log there problem through
image & it's some description after analysis they will get there solution.
If farmer face internet issue then they can call in offline mode & get there
problems solution. There is multi language's are available for user to
interact or log there problem.
